# Projeto AirAnC

Mobile construido com React Native utilizando Expo.

## Tecnologias Utilizadas

* React Native
* Yarn
* Expo
* Axios
* Socket.io

## Instalação

```bash
# Para instalar as dependencias do projeto
yarn install
# Executar Projeto
yarn start
```
